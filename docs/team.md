## Contribution Conventions

1. Create a branch from `develop` with the format `feature/<description>`.
2. Write descriptive commits in English or Spanish (consistent per PR).
3. Ensure `mvn test` passes before opening a Pull Request.
4. Update documentation in `docs/` when modifying visible behavior.
